# AgentOps 모니터링 화면 기능 갭 분석

**분석 일자**: 2025-11-18 (업데이트)  
**범위**: `/admin/monitoring` 화면만 (Today, Agent, Chat 등 다른 화면 제외)  
**목적**: Agent Portal 모니터링 화면과 AgentOps의 기능 차이 분석 및 개선 로드맵 수립

## ⚠️ 중요: 범위 제한

- **개선 대상**: `/admin/monitoring` 페이지만
- **유지 대상**: Today, Agent, Chat 등 기존 Glassmorphism 디자인 화면
- **전역 스타일**: tailwind.config.js 전역 변경 최소화
- **모니터링 전용**: 색상, 폰트, 레이아웃은 모니터링 화면에만 적용
